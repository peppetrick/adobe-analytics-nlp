Utente
Keyword rilevate:
"errore ERR_AUTH_001" → filterDimension eVar56
"pagine in cui" → dimension eVar5
"ultimi 2 giorni" → last2days

Adobe Analytics API Request (step 2/2)
{
  "rsid": "myreportsuite",
  "globalFilters": [{
    "type": "dateRange",
    "dateRange": "2026-03-03.../2026-03-04..."
  }],
  "metricContainer": {
    "metrics": [{
      "id": "metrics/event56",
      "filters": ["filter_err"]
    }],
    "metricFilters": [{
      "id": "filter_err",
      "type": "breakdown",
      "dimension": "variables/evar56",
      "itemId": "<itemId di ERR_AUTH_001>"
    }]
  },
  "dimension": "variables/evar5",
  "settings": {
    "limit": 50,
    "page": 0
  }
}

Tutti i date range vengono convertiti da stringa testuale (es. last2days) in formato ISO da parseDateRange() in adobe.service.js
